## 📌 Descripción

Aplicación interactiva desarrollada en Streamlit que integra los fundamentos de programación en Python aplicados al área de **Sistemas Financieros**.

---

## 📂 Estructura del Proyecto

```
├── app.py                            # Aplicación principal
├── libreria_funciones_proyecto1.py   # Librería de funciones (proporcionada)
├── libreri_a_clases_proyecto1.py     # Librería de clases (proporcionada)
├── requirements.txt                  # Dependencias
└── README.md                         # Este archivo
```

---

## 🧩 Secciones de la Aplicación

| Sección | Descripción |
|---|---|
| 🏠 Home | Presentación del proyecto y del estudiante |
| 📋 Ejercicio 1 | Flujo de caja con listas: registro de ingresos y gastos |
| 📊 Ejercicio 2 | Registro de inversiones con arrays NumPy y DataFrame |
| 🧮 Ejercicio 3 | Calculadora de Valor Futuro con histórico |
| 🏗️ Ejercicio 4 | CRUD de Proyectos de Inversión (VPN, ROI, Payback) |

---

## 🚀 Cómo ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```
